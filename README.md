## 🚀 Sobre Mim

Olá! Sou aluna da **Rede Estadual do Paraná** e completamente apaixonada por tecnologia. 🖥️✨

Para mim, programar é como ter um superpoder: a capacidade de criar coisas incríveis do zero usando apenas linhas de código. Se você estuda no Ensino Fundamental 2 (do 6º ao 9º ano) e acha que tecnologia é coisa de outro mundo, estou aqui para te mostrar que aprender a programar pode ser muito divertido e abrir portas para o futuro!

* 👀 **Meu foco:** Aprender novas linguagens de programação, evoluir meus projetos e compartilhar conhecimento com outros estudantes.
* 💡 **Filosofia:** Errar um código faz parte do aprendizado. O segredo é testar, corrigir e nunca desistir!

## 🛠️ O que estou estudando (Minhas Ferramentas de Criação)

Para construir sites e dar vida às ideias, uso as principais tecnologias do mercado, adaptadas para a nossa paleta estilizada de vermelho e preto:

### 🌐 Criação e Estilo de Páginas
* ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white) — É o "esqueleto" de qualquer site na internet.
* ![CSS3](https://img.shields.io/badge/CSS3-CC0000?style=for-the-badge&logo=css3&logoColor=white) — É o que dá estilo, cores e deixa o visual bonito (como as nossas cores favoritas!).

### 🧠 Lógica e Programação na Escola
Atualmente, junto com meus colegas de sala de aula, estou dominando estas ferramentas:
* ![Scratch](https://img.shields.io/badge/Scratch-D43F3F?style=for-the-badge&logo=Scratch&logoColor=white) — Programação em blocos, perfeita para entender a lógica como se fosse um quebra-cabeça.
* ![JavaScript](https://img.shields.io/badge/JavaScript-222222?style=for-the-badge&logo=javascript&logoColor=F7DF1E) — A linguagem que traz movimento, interatividade e inteligência para os projetos.
* ![GitHub](https://img.shields.io/badge/GitHub-000000?style=for-the-badge&logo=github&logoColor=white) — Minha biblioteca digital onde guardo e organizo todos os meus códigos.
